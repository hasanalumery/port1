Hoi! Ik ben Hasan, een software development student met een passie voor heldere, gebruiksvriendelijke front-ends die écht werken.

De afgelopen 3 jaar bouwde ik uiteenlopende projecten: van een e-commerce shop tot een real-time chatapp. Ik focus op React en Next.js, met schone code en responsive design.

Mijn aanpak draait om gebruiksvriendelijkheid en performance. In agile teams combineer ik daily stand-ups, pair programming en een vleugje fun.

Buiten studie experimenteer ik met frameworks, schrijf ik blogposts over webtrends en blijf ik leren via online cursussen.

Of je nu een landingpage, een dashboard of een eerste webapplicatie zoekt: ik denk graag mee. Let’s build awesome together!
